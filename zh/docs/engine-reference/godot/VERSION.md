## 知识空白警告

LLM 的训练数据对 Godot 的覆盖大约截止到 ~4.3 版本。4.4、4.5、4.6 引入了模型**不知道**的重要变更。在建议任何 Godot API 调用之前，务必交叉核对本目录。

## 截止日期后的版本时间线

| 版本 | 发布时间 | 风险级别 | 关键主题 |
|------|---------|---------|---------|
| 4.4 | ~2025 年中 | 中等 | Jolt 物理可选、FileAccess 返回类型、着色器纹理类型变更 |
| 4.5 | ~2025 年底 | 高 | 无障碍（AccessKit）、可变参数、@abstract、着色器预烘焙、SMAA |
| 4.6 | 2026 年 1 月 | 高 | Jolt 设为默认、辉光重做、Windows 默认 D3D12、IK 系统恢复 |
